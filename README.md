## `Installing`
```bash
# downloading the project
git clone https://github.com/OmarAlaraby/Hotel-Reservation-System.git
```

intstall pipenv
```bash
pip install pipenv
```

go to the project directory and run these commands

```bash
# activate the virtual environment
pipenv shell
```

```bash
pipenv install Pipfile
```

```bash
# run the file
python views.py
```
